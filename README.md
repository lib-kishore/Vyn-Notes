## 🚀 Overview

**VynNotes** is a lightweight, AI-powered note-taking application built for simplicity and speed.  
It is designed for personal productivity—focusing only on essential features without unnecessary complexity.

---

## 💡 Motivation

While using AI tools to generate and structure notes, switching between multiple platforms became inefficient.

Tools like Notion are powerful, but:
- Not fully free for extended use
- Feature-heavy for simple personal workflows

**VynNotes** was created as a minimal alternative—focused on what actually matters.

---

## ⚙️ Tech Stack

- **Python** – Core development language  
- **Streamlit** – UI framework (fast, minimal, built-in theming, easy deployment)  
- **MongoDB** – Flexible database for storing notes  
- **Groq API** – AI integration (fast responses, efficient token usage)

---

## ✨ Features

- AI-assisted note generation  
- Clean and minimal interface  
- Basic text organization (bullets, structure, etc.)  
- Dark / Light mode support  
- Quick deployment using Streamlit  

---

## ⚠️ Limitations

- Limited Markdown support (kept simple by design)  
- No complex workspace or collaboration features  
- Minimal customization options  

> This project intentionally avoids feature bloat.
